### 本仓库用于存放博客相关文件

目录结构：
```
file----------------------------父文件夹
 │
 ├─static-----------------------静态文件父目录(图片、静态js、css等)
 │   │ 
 │   └─images-------------------图片父目录(头像等公共图片)
 │       │
 │       └─pictures-------------存放壁纸、照片等的目录
 │
 └─upload-----------------------上传文件父目录
     │ 
     ├─attachements-------------上传的文件
     │ 
     └─pic----------------------blog中使用的图片
```
