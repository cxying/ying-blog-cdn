# 本仓库用于存放博客相关文件
# 欢迎访问：yingserver.cn

目录结构：
```
file----------------------------父项目
 │
 ├─static-----------------------静态文件(图片、静态js、css等)
 │   │ 
 │   └─images-------------------图片父目录(头像等)
 │       │ 
 │       └─pictures-------------图片(壁纸、照片等)
 │
 └─upload-----------------------上传文件父目录
     │ 
     ├─attachements-------------上传的文件
     │ 
     └─pic----------------------blog中使用的图片
```
